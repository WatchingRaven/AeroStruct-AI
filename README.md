# AeroStruct-AI
Open-source Python-система, которая на основе телеметрии (вибрация, температура, давление, акустическая эмиссия) предсказывает остаточный ресурс (RUL), выявляет аномалии и генерирует ранние предупреждения о рисках
Проект создан инженером-испытателем для демонстрации applied AI в промышленности. Цель: предсказание отказов турбин/ГТД с объяснимой аналитикой»
